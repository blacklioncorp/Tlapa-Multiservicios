# **App Name**: Tlapa Multiservices

## Core Features:

- Geo-Referenced User Verification: Allows field technicians to verify and correct user locations on an interactive map.
- Multiservice Payment Tracking: Tracks payments for water, property taxes (predial), and cadastral services, displaying a payment status on an interactive map.
- Interactive Map Interface: Provides a map-based interface for technicians to select users for verification and correction.
- Automated Status Overrides: Enables administrators to apply special statuses (e.g., mora, desconexi√≥n) to a service and temporarily override the automated payment status (tool).
- Dashboard KPIs and Visualizations: Presents key performance indicators such as total contributors, properties, current services percentage, and revenue through graphs.
- Payment Management: Offers an interface for administrators and cashiers to register payments and automatically update the service status on the map.
- Custom Reporting & Export: Allows generation of reports (PDF and Excel) filtered by location, payment status, and service type, enabling easy extraction of payment data for reconciliation.

## Style Guidelines:

- Primary color: Olive green (#5D8A27), drawing from the green in the Tlapa logo for a natural and municipal feel.
- Background color: Light green (#E4EFE1), a lighter tint of the olive to provide contrast.
- Accent color: Gold (#D4A743) echoing the yellow/gold from the Tlapa logo, will be used for highlighting key actions.
- Font Pairing: 'Poppins' (sans-serif) for headings, 'PT Sans' (sans-serif) for body text.
- Use map-based icons related to services such as water drop, building, and document, based on Google Maps icons.
- Mobile-first design with a clear hierarchy; the mobile app features a prominent map for technicians.
- Subtle transitions for map marker updates and payment status changes for clarity and feedback.
